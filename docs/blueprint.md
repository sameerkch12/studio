# **App Name**: Delivery Tracker Pro

## Core Features:

- Daily Data Upload: Allows daily upload of delivery boy data including parcels delivered, returns, COD collected, and RVP pickups.
- Delivery Tracking: Keeps track of the number of items delivered and any returns, and other key parameters.
- COD Collection Monitoring: Provides precise details for collected COD amounts.
- RVP (Return to Vendor Pickup) Log: Enables efficient recording of return-to-vendor pickups.
- Payment Calculation: Automatic computation of payments to delivery personnel, set at ₹14 per parcel, factoring in earnings of ₹19 per parcel.
- Earnings Report Generation: Generates reports on delivery boy earnings (based on ₹14 per delivery) versus company earnings (based on ₹19 per delivery).

## Style Guidelines:

- Primary color: Deep sky blue (#3091e0) for trust and clarity.
- Background color: Very light blue (#e7f3ff).
- Accent color: Teal (#30c9e0), which is used in charts.
- Body and headline font: 'PT Sans' for a clean, modern look.
- Simple, professional icons for categories like 'Delivered', 'Returns', and 'COD'.
- Clean, tabular layout for displaying data with clear distinctions between fields.
- Subtle transition effects on data updates.
